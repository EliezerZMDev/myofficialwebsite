# project.md — Propósito y Visión del Proyecto

## ¿Qué estamos construyendo?
**Portfolio Personal Oficial** — Sitio web de portafolio interactivo con scroll-driven animations, transiciones suaves entre secciones y efectos visuales inspirados en experiencias web premium como Wuthering Waves. El sitio mostrará habilidades técnicas, proyectos destacados y proporcionará medios de contacto.

## Stack tecnológico
- HTML5 Semántico
- CSS3 (Flexbox, Grid, Variables CSS, Animaciones)
- JavaScript ES6+ (Vanilla JS inicialmente, con planes para integrar React/Vue para efectos complejos como parallax y transformaciones 3D)
- Posible uso de bibliotecas ligeras como GSAP o ScrollTrigger para animations avanzadas

## Objetivos
1. Presentar trabajo profesional de forma memorable y atractiva visualmente
2. Demostrar habilidades técnicas de frontend mediante la propia implementación del sitio
3. Proporcionar una experiencia de usuario fluida que guía al visitante por las secciones clave
4. Facilitar contacto para oportunidades profesionales mediante formulario y enlaces claros
5. Mantener alto rendimiento (<3s carga inicial) y accesibilidad básica

## Usuarios / Público objetivo
- Reclutadores técnicos y gerentes de contratación
- Clientes potenciales para trabajo freelance o contractual
- Colaboradores en proyectos de código abierto o comerciales
- Comunidad de desarrolladores interesados en técnicas de frontend avanzadas

## Estado actual
Estructura básica de carpetas definida. Próximos pasos: crear index.html semántico, establecer variables CSS básicas, implementar estructura de secciones (inicio, sobre mí, proyectos, contacto, habilidades).
